#Chess app

##Features 
>Working chess logic 
>Frontend runs and can be played by text input

###Install
```
git clone https://github.com/Varjovaras/chess-app.git
npm i
npm run installfront
```

###frontend
```
npm run front
```

###backend
```
npm run backend
```

###tests
```
npm run test
```
#####run a ton of games that work as tests
```
npm run chess
```
